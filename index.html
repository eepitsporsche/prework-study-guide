Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element defines metadata such the character set, description, keywords, author, and viewport.
  • "Child elements" are nested within a "parent element".
  • An "attribute" defines additional information about an element.

Git

  • "git status" checks what branch we are currently on.
  • "git checkout -b feature branch name" creates a new branch and switches to it.
  • "git add -A" adds code from the main branch to the current working branch.
  • "git pull origin main" checks if the local branch is synced with the base branch on Github.
  • "git push origin feature branch name" pushes changes to the remote branch on Github.
